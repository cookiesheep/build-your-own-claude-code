# 实验指南

本项目包含 5 个渐进式实验（Lab 0 - Lab 4）。

每个 Lab 包含：

- **知识讲解** — 你需要理解的概念和原理
- **实验任务** — 带 `TODO` 标记的骨架代码，由你补全
- **自动测试** — `npx vitest run` 验证你的实现
- **可视化 Demo** — `npx tsx labs/lab-XX/demo.ts` 展示你的成果
- **渐进式提示** — 卡住时可以查看 hints
- **参考实现** — 完成后对比学习

## 学习路线图

```
Lab 0: 环境搭建 + 体验 Claude Code
  │
  ▼
Lab 1: 消息协议 + LLM API 调用
  │         ← 基础设施，快速完成
  ▼
Lab 2: 工具定义 + 工具执行
  │         ← 核心前置
  ▼
★ Lab 3: Agent Loop（核心循环）
  │         ← 项目核心，80% 的精力在这里
  ▼
Lab 4: 插入 Claude Code
              ← 终极验证
```

## 反馈机制

!!! note "每个 Lab 都有两种反馈"

    1. **单元测试**：使用 Mock LLM（预录的固定响应），不需要 API Key，在任何机器上结果一致
    2. **Demo 脚本**：可视化展示 Lab 成果。没有 API Key 也能运行（自动使用 Mock 数据）
