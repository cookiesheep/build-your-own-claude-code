## 项目背景

**BYOCC** 教学平台。学习者编辑代码提交到 Docker 容器中构建运行。

本任务是**多文件编辑的前端部分**。后端多文件 API（injectFiles、多文件 submit、多文件 workspace）已由上一个开发任务完成。

### 当前状态

- `file-read-api` 已完成：只读文件查看、CodeEditor readOnly/language/loading、FileTree onFileSelect(path, isEditable)
- 后端多文件基础设施已完成：`injectFiles`、多文件 submit、多文件 workspace API
- `file-tree-data.ts` 已有 `LAB_FILES` 配置和 `buildLabFileTree()` 自动生成树

### 目标

改造前端为多文件编辑模式：
- 一个 Lab 有多个可编辑文件（如 Lab 0 有 6 个文件）
- 用户在文件树中切换文件，每个文件独立编辑
- 自动保存所有文件到 workspace API
- 提交时发送所有文件

### 关键设计：编辑不需要容器

- 可编辑文件内容来自 **LAB_FILES 骨架** + **workspace 已保存内容**
- 不需要启动容器就能编辑
- 容器只在提交构建和运行时需要
- 只读文件查看仍需要容器运行（file-read-api 已实现）

### 任务 1：多文件 Workspace + Submit API 客户端

**文件**：`platform/src/lib/api.ts`

#### 1.1 修改 getWorkspace

当前返回 `{ code: string | null, updatedAt: string | null }`。
改为返回 `{ files: Record<string, string>, updatedAt: string | null }`。

```typescript
export type WorkspaceFiles = Record<string, string>;

export async function getWorkspace(labId: number): Promise<{
  files: WorkspaceFiles;
  updatedAt: string | null;
}> {
  const response = await authorizedFetch(apiUrl(`/api/labs/${labId}/workspace`));
  if (!response.ok) {
    return { files: {}, updatedAt: null };
  }
  const data = await response.json();

  // 兼容旧版单文件格式：后端可能返回 { code: "..." } 或 { files: {...} }
  if (typeof data.code === 'string' && !data.files) {
    return { files: { 'src/query-lab.ts': data.code }, updatedAt: data.updatedAt ?? null };
  }

  return {
    files: data.files ?? {},
    updatedAt: data.updatedAt ?? null,
  };
}
```

#### 1.2 修改 saveWorkspace

```typescript
export async function saveWorkspace(
  labId: number,
  files: WorkspaceFiles
): Promise<{ files: WorkspaceFiles; updatedAt: string }> {
  const response = await authorizedFetch(apiUrl(`/api/labs/${labId}/workspace`), {
    method: 'PUT',
    headers: { 'Content-Type': 'application/json' },
    body: JSON.stringify({ files }),
  });
  if (!response.ok) {
    throw new Error('Failed to save workspace');
  }
  const data = await response.json();
  return {
    files: data.files ?? files,
    updatedAt: data.updatedAt ?? new Date().toISOString(),
  };
}
```

#### 1.3 新增 submitFiles

```typescript
export async function submitFiles(
  sessionId: string,
  files: WorkspaceFiles,
  labId: number
): Promise<{ success: boolean; buildLog: string }> {
  const response = await authorizedFetch(apiUrl('/api/submit'), {
    method: 'POST',
    headers: { 'Content-Type': 'application/json' },
    body: JSON.stringify({
      sessionId,
      files,
      labNumber: labId,
    }),
  });
  const data = await response.json();
  return {
    success: data.success ?? false,
    buildLog: data.buildLog ?? '',
  };
}
```

#### 1.4 保留向后兼容的 submitCode

保留原有的 `submitCode` 函数不删（可能有其他地方引用），但标记为 deprecated：

#### 2.2 Bootstrap 改造

```typescript
useEffect(() => {
  let cancelled = false;

  async function bootstrap() {
    try {
      setSaveState("loading");
      setFiles({});
      setActiveFilePath("");
      setViewingFile(null);
      setReadOnlyContent(null);
      setReadOnlyError(null);
      setReadOnlyLoading(false);
      readOnlyRequestIdRef.current += 1;

      await ensurePlatformIdentity();

      const existingSessionId = window.localStorage.getItem(SESSION_STORAGE_KEY) ?? undefined;
      const session = await createSession(existingSessionId);
      if (cancelled) return;

      window.localStorage.setItem(SESSION_STORAGE_KEY, session.sessionId);
      setUserId(session.userId ?? "");
      setSessionId(session.sessionId);
      setEnvironmentStatus(session.environmentStatus);
      setEnvironmentMessage("");

      if (session.environmentStatus === "running") {
        const environment = await getEnvironmentStatus(session.sessionId);
        if (!cancelled) {
          setEnvironmentStatus(environment.environmentStatus);
          setTerminalUrl(environment.success ? (environment.terminalUrl ?? "") : "");
          setEnvironmentMessage(environment.message ?? "");
        }
      }

      // 加载 workspace
      const workspace = await getWorkspace(lab.id);
      if (cancelled) return;

      // 获取当前 Lab 的文件定义
      const labFileDefs = LAB_FILES[lab.id] ?? [];

      // 构建 files map：workspace 有已保存内容就用它，否则用骨架
      const initialFiles: Record<string, string> = {};
      for (const fileDef of labFileDefs) {
        initialFiles[fileDef.path] = workspace.files[fileDef.path] ?? fileDef.skeleton ?? "";
      }
      setFiles(initialFiles);

      // 设置默认活跃文件为第一个可编辑文件
      const editablePaths = labFileDefs.filter(f => f.editable).map(f => f.path);
      if (editablePaths.length > 0) {
        setActiveFilePath(editablePaths[0]);
      }

      setLastSavedAt(workspace.updatedAt);
      setSaveState(workspace.updatedAt ? "saved" : "idle");
    } catch (error) {
      if (!cancelled) {
        const message = error instanceof Error ? error.message : "Unknown session error";
        setEnvironmentStatus("error");
        setEnvironmentMessage(message);
        setSaveState("error");
      }
    }
  }

  void bootstrap();
  return () => { cancelled = true; };
}, [lab.id]);
```

**关键变化**：
- 不再依赖 `LAB_SKELETONS`（已删除）
- `initialFiles` 从 `LAB_FILES` 配置构建
- workspace 已保存的内容优先于骨架

#### 2.4 文件切换

```typescript
const handleFileSelect = async (path: string, isEditable: boolean) => {
  if (isEditable) {
    // 切换到可编辑文件
    setViewingFile(null);
    setReadOnlyContent(null);
    setReadOnlyError(null);
    setReadOnlyLoading(false);
    setActiveFilePath(path);
    return;
  }

  // 只读文件：从容器读取
  if (environmentStatus !== "running") {
    setReadOnlyError("请先启动实验环境再查看文件");
    return;
  }

  setViewingFile(path);
  const requestId = readOnlyRequestIdRef.current + 1;
  readOnlyRequestIdRef.current = requestId;
  setReadOnlyLoading(true);
  setReadOnlyError(null);
  setReadOnlyContent(null);
  try {
    const result = await fetchFileContent(path, sessionId);
    if (readOnlyRequestIdRef.current !== requestId) return;
    setReadOnlyContent(result.content);
    setReadOnlyLanguage(result.language);
  } catch (error) {
    if (readOnlyRequestIdRef.current !== requestId) return;
    setReadOnlyError(error instanceof Error ? error.message : "读取文件失败");
  } finally {
    if (readOnlyRequestIdRef.current === requestId) {
      setReadOnlyLoading(false);
    }
  }
};
```

#### 2.5 自动保存改造

```typescript
useEffect(() => {
  if (saveState !== "dirty") return;

  const timer = window.setTimeout(() => {
    const requestId = saveRequestIdRef.current + 1;
    saveRequestIdRef.current = requestId;
    setSaveState("saving");
    void saveWorkspace(lab.id, files)
      .then((workspace) => {
        if (saveRequestIdRef.current === requestId) {
          setLastSavedAt(workspace.updatedAt);
          setSaveState("saved");
        }
      })
      .catch(() => {
        if (saveRequestIdRef.current === requestId) {
          setSaveState("error");
        }
      });
  }, 1500);

  return () => { window.clearTimeout(timer); };
}, [files, lab.id, saveState]);
```

**变化**：`code` → `files`，其他逻辑不变。
